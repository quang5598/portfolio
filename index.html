English Learning Tools

Summary

The English Learning Tools website offers some useful tools such as the dictionary, sentence examples, vocabulary quiz, etc. Those tools can help English learners improve their language skills further.

Technology Used

The project is built in ASP.NET MVC 5. It consumes several external APIs to get data for different tools. Users can register or login to the website, their data will be stored in a cloud database called Microsoft Azure Cloud Database.

Book Information Page

Summary

Book Information Page is built on a Javascript library called React. The page will fetch information from the Google Books APIs to retrieve book information based on user searches. Users can log in or register to use the Reading List function to store their favorite books.

Technology Used

The project uses node.js and express.js as a back-end server to receive and send back data to the front-end (React). User information will be stored in the database management system called PostgreSQL. Both the back-end server and database were deployed in a cloud platform called Heroku.

Grammar Police

Summary

Whether your English is a second language or a native language, you will always make grammatical mistakes (especially me!). But do not worry, Winnie-the-Police has a duty to find any grammatical errors for you.

Technology Used

The Grammar Police project uses the Grammar Bot API to check user text and find any grammatical errors. The technology used in this project is React as a front-end to transfer and receive data from the back-end server. Express.js (node.js) is used as back-end to receive and return data to the front-end.
